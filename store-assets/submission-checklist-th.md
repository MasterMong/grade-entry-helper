# รายการตรวจสอบการส่งส่วนขยาย Chrome Web Store

## ข้อกำหนดก่อนการส่ง

### 1. การตั้งค่าบัญชีนักพัฒนา
- [ ] สร้างบัญชีนักพัฒนา Chrome Web Store (ค่าธรรมเนียม $5 ครั้งเดียว)
- [ ] ยืนยันตัวตน (อาจต้องใช้บัตรประจำตัวที่รัฐบาลออกให้)
- [ ] กรอกข้อมูลโปรไฟล์นักพัฒนาให้ครบถ้วน

### 2. แพ็คเกจส่วนขยาย
- [ ] รัน `npm run build` สำหรับ production build
- [ ] ทดสอบส่วนขยายในโปรไฟล์เบราว์เซอร์ใหม่
- [ ] ตรวจสอบให้แน่ใจว่าฟีเจอร์ทั้งหมดทำงานถูกต้อง
- [ ] สร้างไฟล์ ZIP จากโฟลเดอร์ `dist/` (ไม่รวมไฟล์ source code)

### 3. ไฟล์สำหรับ Store
- [ ] ไอคอน 128x128px (มีอยู่แล้วใน dist/icons/)
- [ ] ภาพหน้าจอ (1-5 รูป, ขนาด 1280x800 หรือ 640x400px)
- [ ] รูปภาพโปรโมท (ทางเลือก):
  - [ ] Small tile: 440x280px
  - [ ] Large tile: 920x680px
  - [ ] Marquee: 1400x560px

### 4. ข้อมูลการลิสต์
- [ ] ชื่อส่วนขยาย: "SGS Bot"
- [ ] สรุปย่อ (ไม่เกิน 132 ตัวอักษร)
- [ ] คำอธิบายโดยละเอียด
- [ ] หมวดหมู่: Productivity
- [ ] ภาษา: ไทย/อังกฤษ
- [ ] URL เว็บไซต์ (GitHub repo หรือเว็บไซต์โครงการ)
- [ ] อีเมลสำหรับการสนับสนุน

### 5. ข้อกำหนดทางกฎหมาย
- [ ] นโยบายความเป็นส่วนตัว (สร้างแล้วใน privacy-policy.md)
- [ ] ข้อกำหนดการใช้บริการ (หากมี)
- [ ] เตรียมคำอธิบายเหตุผลการขอสิทธิ์ต่างๆ

### 6. การเตรียมตัวสำหรับการตรวจสอบ
- [ ] ทดสอบในหลายขนาดหน้าจอ
- [ ] ตรวจสอบการปฏิบัติตามมาตรฐานการเข้าถึง
- [ ] ตรวจสอบการละเมิดนโยบาย
- [ ] ตรวจสอบให้แน่ใจว่าไม่มีเนื้อหาที่ต้องห้าม

## ขั้นตอนการส่ง

1. **อัปโหลดส่วนขยาย**
   - อัปโหลดไฟล์ ZIP จากโฟลเดอร์ `dist/`
   - Chrome Web Store จะกรอกข้อมูลจาก manifest ให้อัตโนมัติ

2. **กรอกข้อมูลการลิสต์**
   - เพิ่มไฟล์และคำอธิบายทั้งหมด
   - ตั้งราคา (ฟรี)
   - กำหนดค่าการแจกจ่ายตามภูมิภาค

3. **ส่งเพื่อตรวจสอบ**
   - การตรวจสอบใช้เวลา 1-7 วัน
   - ตอบกลับความคิดเห็นจากผู้ตรวจสอบอย่างรวดเร็ว

## หลังการส่ง

- [ ] ติดตามสถานะการตรวจสอบ
- [ ] ตอบกลับความคิดเห็นจากผู้ตรวจสอบหากจำเป็น
- [ ] วางแผนการอัปเดต/บำรุงรักษา
- [ ] ตั้งค่ากระบวนการสนับสนุนผู้ใช้

## หมายเหตุ
- Chrome Web Store ให้ความสำคัญกับส่วนขยายที่มีจุดประสงค์เดียว
- การอธิบายเหตุผลการขอสิทธิ์อย่างชัดเจนช่วยในกระบวนการตรวจสอบ
- ภาพหน้าจอคุณภาพสูงช่วยเพิ่มอัตราการแปลง

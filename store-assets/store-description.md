## Description
**SGS Bot** simplifies grade management for teachers using the Student Grading System (SGS) at sgs.bopp-obec.info.

### Key Features:
- **Bulk Grade Import**: Copy grades from Google Sheets and paste them directly into SGS
- **Dynamic Column Detection**: Automatically adapts to any grading configuration
- **Smart Validation**: Ensures data integrity before submission
- **One-Click Navigation**: Quick access to SGS from browser toolbar
- **Real-time Status**: Live feedback during grade entry process

### How It Works:
1. Copy grade data from your Google Sheets spreadsheet
2. Navigate to the SGS grading page
3. Click the extension's control panel
4. Paste grades with automatic field mapping
5. Submit with confidence

### Perfect For:
- Teachers managing multiple grade columns
- Bulk grade entry from spreadsheet data
- Streamlining repetitive grading tasks
- Reducing manual data entry errors

**Note**: This extension only works with the official SGS system at sgs.bopp-obec.info and requires appropriate access permissions.

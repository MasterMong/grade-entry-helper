# Privacy Policy for SGS Bot

Last updated: [Current Date]

## Data Collection
SGS Bot does not collect, store, or transmit any personal data or user information to external servers.

## Data Processing
- **Clipboard Access**: Used only to read grade data from clipboard for local processing
- **Page Content**: Accessed only on SGS pages to populate grade fields
- **No Data Storage**: No user data is stored locally or remotely
- **No Tracking**: No user analytics, tracking, or profiling

## Permissions Usage
- **clipboardRead**: Read clipboard data for grade import functionality
- **activeTab**: Access current SGS tab for grade field population
- **tabs**: Navigate to SGS pages when extension icon is clicked
- **scripting**: Execute grade entry scripts on SGS pages
- **Host Permission (sgs.bopp-obec.info)**: Required to function on SGS system only

## Data Security
All data processing occurs locally in your browser. No information is transmitted to third parties or stored on external servers.

## Third-Party Services
This extension does not integrate with or send data to any third-party services.

## Children's Privacy
This extension is intended for educational/professional use and does not knowingly collect information from children under 13.

## Changes to Privacy Policy
Any updates to this policy will be reflected in extension updates with clear notification.
